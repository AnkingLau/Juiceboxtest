# Team-5
1. About uploading code files：  
```bash
 1.1 Adding a Remote Repository：% git remote add origin https://github.cs.adelaide.edu.au/MCI-Projects-2024/Team-5.git  
 1.2 Validating remote repositories：% git remote -v  
 1.3 Pulling (fetch) remote repositories：% git fetch origin  
 1.4 Find your own branch  
 1.5 Switch to the target branch：% git checkout -b your_branch origin/your_branch  
 1.6 Add file：  
     % git add your_file_name  
     % git add .(previously uploaded, just changed)  
     % git commit -m "Your commit message"  
 1.7 Pushing Changes to a Remote Branch：git push origin your_branch  
 ```
